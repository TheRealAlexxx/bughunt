Bug Hunt

Bug hunt is a small full-stack game where you fix broken code by picking the right solution.

It’s mainly built as a way to practice debugging and basic programming logic in a simple format

The backend is written in python, and the frontend uses JavaScript, HTML, and css.

Play it here: https://bughunt-gamma.vercel.app/

how it works

Each round shows a piece of buggy code. you pick what you think is the correct fix from a set of options. you get instant feedback, your score updates, and the game moves on to the next question. at the end, you see your final score.
